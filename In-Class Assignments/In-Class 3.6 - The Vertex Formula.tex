\[
\Gp{h,k}=\Gp{\dfrac{-b}{2a},\dfrac{-b^2+4ac}{4a}}
\]
to rewrite each function in vertex form.

\begin{clist}
\item Determine whether the function has a maximum value or a minimum value.
\item What is the maximum or minimum value (whichever exists)?
\item For what value of $x$ does the function obtain this value?
\end{clist}
