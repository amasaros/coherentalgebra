\subsection{Distance From a Line} 
We know how to find the distance between two \makeblank[1in]. How do we find the distance between a point and a line?
\begin{itemize}
\item The distance depends on what \makeblank[1in] on the line we choose.
\item We choose the \makeblank[1in] possible \makeblank[1in].
\item This is on a \makeblank[1in] line.
\end{itemize}
\begin{Example}
How far is the point $(4,3)$ from the line $y=-\frac{3}{4} x - \frac{1}{4}$?

To find the distance without the picture:
\begin{enumerate}
\item Find a perpendicular line through the given point.
\item Find where this intersects the given line.
\item Find the distance between them.
\end{enumerate}
