\subsection{A System of Three Equations by Elimination} 
In general, we'll use elimination to solve a system of more than two equations, unless the system is set up to make substitution exceptionally easy.
\begin{example}
Solve
\[
\begin{systemLa}
2x&+&3y&-&z&=&-2&\\
3x&-&5y&+&2z&=&23&\\
x&+&2y&-&3z&=&-7&
\end{systemLa}
\]

\end{lpic}
The solution is $(x,y,z)=\blanktrip$

The work shown here is the actual work to solve this system---including the notes in words.
\end{note}
\end{example}
