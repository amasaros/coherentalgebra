\subsection{Describing the Solutions of a Dependent System} 
We saw that a dependent system in three variables has a \makeblank[1in] of solutions. Can we say what the solutions are, instead of just ``infinitely many solutions''?
\begin{displaybox}[A line in three dimensions]
We can describe a line in space as an ordered triple containing only one variable, in one of these forms:
\[
(x,y(x),z(x))
\qquad
(x(y),y,z(y))
\qquad
(x(z),y(z),z)
\]
We can choose any variable we like to describe a line. We pick whichever makes the math easy.
\end{displaybox}
\begin{Example}
Solve
\[\begin{systemLa}
4x&-&y&-&3z&=&-3&\\
2x&+&y&-&z&=&1&\\
5x&+&4y&-&2z&=&5&
\end{systemLa}\]
This is a dependent system:

\end{Example}
