\subsection{Inconsistent Systems in Three Variables} 
There are three basic ways that a system in three variables can be inconsistent.
\begin{icpic}[x=\dimtd,y=\dimtd]
\begin{scope}[xshift=-.2\textwidth]
	\foreach \y/\col in {0/pca,0.5/pcb,1/pcc}
		\filldraw[\col,plane] (\project{-1}{\y}{-1}) --
													(\project{1}{\y}{-1}) --
													(\project{1}{\y}{1}) --
													(\project{-1}{\y}{1}) --
													cycle;
\end{scope}
\begin{scope}[xshift=.2\textwidth]
	\foreach \y/\col in {0/pca,0.75/pcb!50!pcc}
		\filldraw[\col,plane] (\project{-1}{\y}{-1}) --
													(\project{1}{\y}{-1}) --
													(\project{1}{\y}{1}) --
													(\project{-1}{\y}{1}) --
													cycle;
\end{scope}														
\end{icpic}
In this figure, all of the planes are \makeblank[1in]. If all of the equations are in \makeblank[1in] form:
\begin{align*}
\begin{systemLa}
2x&+&3y&-&4z&=&5\\
2x&+&3y&-&4z&=&2\\
2x&+&3y&-&4z&=&-1
\end{systemLa}
&&
\begin{systemLa}
2x&+&3y&-&4z&=&2\\
2x&+&3y&-&4z&=&2\\
2x&+&3y&-&4z&=&-1
\end{systemLa}
\end{align*}
If we combine any \makeblanksmall of these equations, we will \makeblank and the resulting equations will be \makeblank[1in].
\begin{icpic}[x=\dimtd,y=\dimtd]
\foreach \y/\col in {0.25/pcb,0.75/pcc}
	\filldraw[\col,plane] (\project{-1.5}{\y}{-1}) --
												(\project{ 1.5}{\y}{-1}) --

												cycle;
\filldraw[pca,plane]  (\project{ 1}{-.5}{-1}) -- 
                      (\project{ 1}{-.5}{ 1}) --
											(\project{-1}{1.5}{ 1}) --
											(\project{-1}{1.5}{-1}) --
											cycle;
\foreach \x/\y/\col in {.25/.25/pca!50!pcc,-.25/.75/pcb!50!pcc} 
	\draw[\col,plane] (\project{\x}{\y}{-1}) -- (\project{\x}{\y}{1});
\end{icpic}
Here \emph{two} of the planes are \makeblank[1in], so they intersect the third plane at \makeblank[1in] lines. An example of planes like this, written in \makeblank[1in] form:
\[\begin{systemLa}
3x&-&5y&+&2z&=&7\\
3x&-&5y&+&2z&=&9\\
2x&+&8y&-&3z&=&4
\end{systemLa}\]
We see \makeblanksmall equations that, if we combine them, we will \makeblank[1.5in] and the resulting equations will be \makeblank[1in].
