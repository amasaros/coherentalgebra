\subsection{Complications in Substitution} 
This system is also set up to make substitution easy, but we must be careful.
\begin{example}
Solve
\[\begin{systemL}
x=2z&\\
y=x-5&\\
2x-3y+z=9&
\end{systemL}\]
\begin{itemize}
\item All three equations include \makeblanksmall
\item Since it appears in all three equations, we'll replace \makeblanksmall \emph{last}.
\end{itemize}
\begin{lpic}{}{8}
\end{lpic}
The solution is $(x,y,z)=\blanktrip$
\end{example}
The next example is more confusing than difficult.
\begin{example}
Solve
\[\begin{systemL}
y=1-2x&\\
z=2y-8&\\
x=\frac{1}{3}z-5&
\end{systemL}\]
Note that, while each equation contains \makeblanksmall variables, the entire system contains \makeblanksmall variables.

\end{lpic}
The solution is $(x,y,z)=\blanktrip$
\end{example}
