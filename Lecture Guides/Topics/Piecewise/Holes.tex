\subsection{Holes in a Graph} 
Sometimes, a piecewise graph has just one special point.
\begin{example}
Sketch the graph of $f(x)=\pw{x^2-2&x&\neq&1\\3&x&=&1}$
\begin{itemize}
\item First, we graph the main function (the one that applies for \makeblank[1in] values of $x$)
\item Then, we make a ``hole'' in the graph at $x=\makeblanksmall$.
\item Finally, we plot the special point.
\end{itemize}

\end{example}
