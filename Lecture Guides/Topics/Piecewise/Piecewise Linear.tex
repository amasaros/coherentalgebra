\subsection{Graphing a Piecewise Linear Function} 
The simplest piecewise functions are \emph{piecewise linear} functions--where each piece is linear.
\begin{Example}
Sketch the graph of $f(x)=\pw{2x-1&-2&\leq&x&<&-1\\3-x&-1&\leq&x&\leq&2\\\frac{1}{2}x+1&2&<&x&\leq&4}$
\begin{itemize}
\item We imagine cutting the graph into vertical ``strips'' corresponding to each \makeblank[1in] of the \makeblank[1in].
\item Each piece of this graph is a \makeblank[1.5in], so we need \makeblanksmall points to sketch it.
\item We choose the edges of the strips.
\item If the point is included in the graph, we draw a \makeblank[1in] circle there.
\item If the point is not included, we draw an \makeblank[1in] circle there.
\end{itemize}
