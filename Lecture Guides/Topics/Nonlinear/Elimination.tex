\subsection{Solving a Non-Linear System by Elimination} 
It is usually not possible to solve a nonlinear system by elimination.
\begin{Example}
\[\begin{systemL}
x^2+xy+y^2=7&\\
xy=2&
\end{systemL}\]
We \emph{cannot} solve this by elimination, because the second equation doesn't have \makeblanksmall or \makeblanksmall terms. We must solve this by \makeblank[1in].

This system has \makeblanksmall solutions!
\end{Example}
