\subsection{A Challenging Example} 
When we meet a nonlinear system, we should:
\begin{enumerate}
\item Use \makeblank[1.5in] if possible (it'll make our lives easier!)
\item Use \makeblank[1.5in] if one of the equations can be solved for one variable (sometimes it can't!)
\item Use \makeblank[1.5in] if both of those fail.
\end{enumerate}
\begin{Example}

\[\begin{systemL}
x^2+xy+y^2=19&\\
y^2-x^2=5&
\end{systemL}\]

\end{Example}
