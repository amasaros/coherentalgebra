\subsection{Logs with Base Less than 1} 
How does the graph of a logarithm differ when the base is less than 1?
\begin{Example}
Sketch the graph of $f(x)=\log_{1/2}(x)$
\begin{lpic}{}{10}
	\draw (1,-2) -- (5,-2);
	\draw (3,-1) -- (3,-9);
	\regtext{2}{-2}{$x$};
	\regtext{4}{-2}{$y$};
	\foreach \x in {-3,...,3}
		\regtext{4}{-\x-6}{$\x$};
\end{lpic}
\end{Example}
Like with the \makeblank[1.5in] graphs, the only change is to the end behavior.
\begin{itemize}
\item The domain is still \makeblank[1in].
\item The range is still \makeblank[1in].
\item There is still a \makeblank[1in] asymptote at \makeblank[1in].
\item But now the end behavior is:
\begin{itemize}
\item When $x\rightarrow+\infty$, $f(x)\rightarrow\makeblanksmall$.
\item When $x\rightarrow\makeblanksmall$, $f(x)\rightarrow\makeblanksmall$.
\end{itemize}
\end{itemize}
