\subsection{Pretend Polynomials with Exponential Terms} 
There are a few cases in which we can solve an equation containing exponential expressions and more than two terms by pretending we have a polynomial.
\begin{example}
Solve $4^x+4=5\cdot 2^x$.
\begin{itemize}
\item We notice that $4=\makeblanksmall$
\item So $4^x=\makeblanksmall=\makeblanksmall=\makeblanksmall$
\end{itemize}
