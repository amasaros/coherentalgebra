\subsection{Basic Exponential Graphs}
What does the graph of an exponential function look like?
\begin{example}
Sketch the graph of $f(x)=2^x$
\begin{lpic}{}{10}
	\draw (1,-2) -- (5,-2);
	\draw (3,-1) -- (3,-9);
	\regtext{2}{-2}{$x$};
	\regtext{4}{-2}{$y$};
	\foreach \x in {-3,...,3}
		\regtext{2}{-\x-6}{$\x$};
\end{lpic}
\end{example}
We see several things in this shape.
\begin{itemize}
\item The domain is \makeblank.
\item The range is \makeblank.
\item We can also describe the end behavior:
\begin{itemize}
\item When $x\rightarrow+\infty$, $f(x)\rightarrow\makeblanksmall$.
\item When $x\rightarrow-\infty$, $f(x)\rightarrow\makeblanksmall$.
\end{itemize}
\item The dashed red line shows that the graph gets close to \makeblanksmall but never touches it. This line is called an \emph{asymptote}. This is a \makeblank[1in] asymptote at \makeblank[1in].
\end{itemize}
We will see the same basic shape as long as \makeblank[1in].
