\subsection{Basic Exponential Equations} 
Because we don't know the \makeblank[1.5in] of $f(x)=b^x$, solving exponential equations can be challenging at this point. However, we can solve them in a special case.
\begin{example}
Solve $5^x=125$

\end{example}
We can generalize this method to let us solve any exponential equation, as long as we can rewrite each side in the same \makeblank[1in].
\begin{example}
Solve $\left(\frac{1}{5}\right)^{x-5}=25^{3x-1}$
\begin{lpic}{}{10}
\end{lpic}
\end{example}
However, since the \makeblank[1in] of an exponential is not all real numbers, we will sometimes have \makeblank.
\begin{example}
Solve $4^x=-4$.
\end{example}
This equation has no solution, because $4^x\makeblanksmall$ for every $x$.
