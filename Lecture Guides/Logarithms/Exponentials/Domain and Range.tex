\subsection{Properties of Exponential Functions}
We can describe the properties of general exponential functions
\[
f(x)=a(b)^{cx-h}+k
\]
\begin{description}
\item[Domain] The domain of an exponential function is always \makeblank[1.5in].
\item[Range] It's always the case that $b^{cx-h}\makeblank[1in]$. So we can find the range of any exponential function.
\end{description}

\end{lpic}
