\subsection{Introduction to Exponential Functions} 
\begin{definition}[Exponential Function]
An \term{exponential function} is a function of the form
\[
f(x)=b^x
\]
where $b>0$ and $b\neq 1$. The number $b$ is called the \term{base}.
\end{definition}
Why do we require $b>0$ and $b\neq 1$?
\begin{itemize}
\item If $b=\makeblanksmall$, $b^x$ is undefined when $x$ is \makeblank[1in] or \makeblanksmall.
\item If $b$ is \makeblank[1in], $b^x$ is undefined when $x$ is a \makeblank[1in] with an even \makeblank[1.5in].
\item If $b=\makeblanksmall$, $b^x=\makeblanksmall$ for \makeblank[1.5in] of $x$.
\item So $b^x$ is only defined (and interesting!) for \makeblank when \makeblank.
\end{itemize}

Like other functions, we can compose an exponential with \makeblank[1in] functions. The following are exponential functions:
\[
f(x)=2\cdot 3^{2x-3}+1
\qquad
g(x)=\left(\tfrac{2}{3}\right)^{x+5}-7
\qquad
h(x)=-4^{x-4}+2
\]
