\section{Rational Roots Theorem}
We now know that if we can factor a polynomial, we can find the roots, and if we can guess the roots, we can find the factors. In practice, we combine these techniques: we guess some of the factors, until we get to a polynomial ``small'' enough to use our basic techniques from \unref{UnPolyBasics}.

We could guess the roots by randomly plugging in numbers, but we would have to get lucky, and we could never know when we were done looking. In this section, we will learn how to come up with a list of all possible \emph{rational} roots of a polynomial with rational coefficients, and to use this to find all possible factors with rational coefficients.
