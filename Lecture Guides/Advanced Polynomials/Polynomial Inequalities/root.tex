\section{Polynomial Inequalities} 
We now know everything we need to know in order to find, at least the rational solutions to polynomial equations. What about inequalities?

We haven't discussed inequalities in a long time---not since \unref{UnAbsVal}! That's because, in general, non-linear inequalities have some complexities that we don't want to get too tangled with. But the techniques we use with polynomial inequalities will be very useful in general, when we work with more complicated functions in the future.
