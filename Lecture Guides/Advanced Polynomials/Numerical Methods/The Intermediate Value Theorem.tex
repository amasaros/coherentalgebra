\subsection{The Intermediate Value Theorem} 
\begin{theorem}[Intermediate Value Theorem]
For a continuous function $f$ (such as a polynomial), if we have two numbers $a<b$ such that one of:
\begin{itemize}
\item $f(a)<0$ and $f(b)>0$
\item $f(a)>0$ and $f(b)<0$
\end{itemize}
there is a number $c$ such that $a<c<b$ and $f(c)=0$
\end{theorem}
\begin{itemize}
\item This theorem says that if a continuous function $f$ changes \makeblank[1in], it must pass through zero in between.
\item In other words, it says that when the computer draws that connected curve, it's justified.
\item What makes a function continuous? \makeblank
\item For now, we know that \makeblank functions are continuous.
\end{itemize}
