\section{Numerical Methods} 
There was a big ``if'' in our discussion of finding the roots of a polynomial (that you might have noticed if you made a mistake in copying coefficients). We can fully factor a polynomial with irrational factors... \emph{if} we can get down to a quadratic factor. What can we do if we can't?

There are tricks and maneuvers that \emph{can} help us, but in the end, we need to decide what it is we really want to know. If we just want a decimal approximation of the zero, we can use what we call ``numerical methods'' to guess the value as closely as we like.
