\subsection{Roots, Zeros, and Factors}
There is a little bit of vocabulary that we're going to need in the rest of this unit.
\begin{definition}[Root]
For a polynomial $$\sum_{k=0}^n{a_k x^k}$$ a \term{root} $r$ of the polynomial is a number such that $$\sum_{k=0}^n{a_k r^k}=0$$
\end{definition}
\begin{definition}[Zero]
For a function $f$, a \term{zero} $z$ of the function is a number such that $$f(z)=0$$
\end{definition}
Notice, if we have a polynomial function $P(x)$, its roots and its zeros are the same. The difference is:
\begin{itemize}
\item A \makeblank has roots
\item A \makeblank has zeros
\end{itemize}
When we're dealing with polynomial functions, we sometimes get lazy with these terms and use them interchangeably.

This last term we already know, but now we are going to be explicit about it.
\begin{definition}[Factor of a Polynomial]
A \term{factor} of a polynomial $P$ is any polynomial $F$ for which there is a polynomial $Q$ such that $$P=Q\cdot F$$
In other words, if a polynomial is written as a product of two other polynomials, those other polynomials are called factors.
\end{definition}
