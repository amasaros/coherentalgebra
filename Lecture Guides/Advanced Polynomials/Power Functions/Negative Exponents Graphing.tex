\subsection{Graphs of Negative Integer Powers} 
Consider these example graphs.
\begin{center}
\begin{tikzpicture}[x=0.5\linespace,y=0.5\linespace]
\setgraphsmall
\useasboundingbox (.50\textwidth,-\value{graphsize}) rectangle (.75\textwidth,\value{graphsize});
\foreach \sh in {1,...,4}
	{
	\begin{scope}[xshift=\sh*0.25*\textwidth]
		\GraphSmall
		\ha{0}
		\va{0}
		\draw[domain={1/\value{graphsize}^(1/\sh)}:\value{graphsize},samples=100,style=graph] 
		      plot (\x,{1/(\x)^(\sh)});
		\draw[domain={1/\value{graphsize}^(1/\sh)}:\value{graphsize},samples=100,style=graph] 
		      plot (-\x,{1/(-\x)^(\sh)});
		\draw (\value{graphsize}+.5,-\value{graphsize}-.5) 
		      node[anchor=south east]{\small$y=x^{-\sh}$};
	\end{scope}
	}
\end{tikzpicture}
\end{center}

\item If $n$ is \makeblank[1in], the two curves are in quadrants \makeblanksmall and \makeblanksmall, passing through \blankpair and \blankpair.

\item The \makeblank[1in] the value of $n$, the \makeblank[1in] and \makeblank[1in] the graph.
\end{itemize}
