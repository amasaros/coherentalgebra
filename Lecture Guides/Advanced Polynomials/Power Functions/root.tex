\section{Power Functions}
In this section, we will look a family of functions related to the polynomials, the \emph{power functions}. We will start by seeing the graphs of power functions, which we will use to understand the graphs of polynomials. Then, we will see how we can apply our knowledge of polynomials to solve equations including several power functions.

Power functions are functions in the form $f(x)=x^p$, where $p$ can be any real number for an appropriate domain. We can extend these into a family by composing them with something linear--in the most general form
$$f(x)=a(cx-h)^p+k$$
These include families of functions we have already seen:
\begin{itemize}
\item Radical functions: $f(x)=a\sqrt[n]{x-h}+k=a(cx-h)^{1/n}+k$
\item Quadratic functions in vertex form: $f(x)=a(x-h)^2+k$
\end{itemize}
This family of functions can be extended to include any number as an exponent, which is what we'll be doing in this section.
