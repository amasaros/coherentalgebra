\subsection{Rewriting with Rational Exponents} 
We have properties of roots that let us combine and separate roots with the same index:
\begin{fact}[Properties of radicals]
For any values of $x$ and $y$ for which all roots below are real numbers,
$$\sqrt[n]{xy}=\sqrt[n]{x}\sqrt[n]{y}
\qquad
\sqrt[n]{\dfrac{x}{y}}=\dfrac{\sqrt[n]{x}}{\sqrt[n]{y}}$$
\end{fact}
For roots with different indexes, we \emph{could} work out a system similar to the one for exponents, but instead, we can just rewrite using rational exponents and use the exponent rules directly.
\begin{example}
Write $\sqrt[3]{x^2y}\sqrt[4]{xy^3}$ as a single radical. Assume all variables represent positive real numbers.

\end{example}
