\subsection{Simplifying Radicals with Multiple Variables} 
If we have a radical expression with multiple variables (and possibly a coefficient), we can just treat each factor separately.
\begin{example}
Simplify $\sqrt[3]{40x^7y^6z^{11}}$
\begin{lpic}{}{5}
\foreach \y/\exp in {1/40,2/x^7,3/y^6,4/z^{11},5/40x^7y^6z^{11}}
	\regtextright{4.25}{-\y}{$\sqrt[3]{\exp}={}$};
\end{lpic}
\end{example}
\noi By factoring them into primes, we can handle coefficients the same way that we do variables. (This can be especially helpful for large coefficients.)
\begin{example}
Simplify $\sqrt{972x^5y^3z^8w}$
\begin{lpic}{}{8}
\foreach \y/\exp in {1/972,4/x^5,5/y^3,6/z^8,7/w,8/972x^5y^3z^8w}
	\regtextright{5.25}{-\y}{$\sqrt{\exp}={}$};
\end{lpic}
\end{example}
\noi Usually, we won't write out each variable separately. Of course, you can write out more detail if it helps you keep track.
\begin{example}
Simplify $\sqrt[4]{18x^7y^2z^{16}w^9}$
\begin{lpic}{}{2}
\regtextleft{0.25}{-1}{$\sqrt[4]{18x^7y^2z^{16}w^9}={}$};
\end{lpic}
\end{example}
