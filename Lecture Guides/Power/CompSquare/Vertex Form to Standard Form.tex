\subsection{Vertex Form to Standard Form} 
Given a quadratic function in vertex form, we can write it in standard form just by simplifying.
\begin{Example}Write the function in standard form and graph.
\begin{lpic}{}{14}
\GraphIn{6}{}
\regtextleft{0.25}{-1}{$f(x)=-\frac{1}{2}(x-2)^2+5$};
\regtext{1}{-6}{$x$};
\regtext{4}{-6}{$y=f(x)$};
\foreach \x in {-6,-4,...,6} 
	\regtextright{1.5}{-10-\x/2}{$\x$};
\draw (0,-6) -- (6,-6);
\draw (2,-5) -- (2,-13);
\end{lpic}
We notice that the vertex is at $\blankpair$. We can see this pair in the vertex form.
\end{Example}
