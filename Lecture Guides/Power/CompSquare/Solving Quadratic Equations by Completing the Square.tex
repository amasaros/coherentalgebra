\subsection{Solving Quadratic Equations by Completing the Square} 
We can use the ``completing the square'' procedure to write quadratic equations in a form in which we can solve using the square root property.
\begin{procedure}{Solve by Completing the Square}
\begin{enumerate*}
\item By adding, subtracting, and multiplying or dividing by numbers, write the equation in the form $x^2+bx=\#$.
\item Complete the square on the left by adding the same constant to both sides.
\item Write the equation in the form $(x+\#)^2=\#$.
\item Solve using the square root property.
\end{enumerate*}
\end{procedure}
