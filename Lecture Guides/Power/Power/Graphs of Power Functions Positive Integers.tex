We observe the following:
\begin{itemize}
\item \makeblank[.75in] exponents give an \makeblanksmall or \makeblanksmall shaped graph.
\item \makeblank[.75in] exponents give a \makeblanksmall or \makeblanksmall shaped graph.
\item Each graph is flat near \blankpair. A \makeblank[1in] exponent is flatter.
\item When $x$ is far away from \makeblanksmall, the graph \makeblank. This is stronger for \makeblank[1in] exponents.
\item If we add a negative sign, the effect is to \makeblank.
\end{itemize}
