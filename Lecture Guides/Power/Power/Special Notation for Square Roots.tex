\subsection{Special Notation for Square Roots} 
We take the root of index \makeblanksmall often enough that it has special notation.
\begin{definition}[Square Root]
The \term{square root} is the root of index 2. We write
$$f(x)=\sqrt{x}$$
to mean the function such that
$$f\inv(x)=x^2,\ x\geq 0$$
Note that there is no index shown for this radical. When no index is shown, the index is \makeblanksmall.
\end{definition}
\begin{note}Historically, the square root predates the other roots. It represents ``the side of a square with the given area.'' The other radical functions generalize this to higher powers (with less geometric reality).\end{note}
\begin{example}
Find each root by rewriting in terms of a power and using trial and error.

	\regtextright{1}{-\y}{\l. };
	\regtextleft{1}{-\y}{$\sqrt{\x}$};
}
\end{lpic}
\end{example}
