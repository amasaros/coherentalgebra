\subsection{Vertex Formula for Vertex Form} 
Instead of manipulating the equation, we can use the vertex formula to write a function in vertex form, for example, in order to graph it.
\begin{example}
Use the vertex formula to rewrite $f(x)=\frac{1}{2}x^2-4x+3$ in vertex form. Then, use the vertex form to sketch the graph of the equation.
