\subsection{Equations with One Radical: Advanced} 
Because power functions are \makeblank[1.5in] functions, we can solve radical equations which also have occurrences of the variable outside the radical.
\begin{note}
In this example, we use an \makeblank[1in] root, so we don't have any worries about the range.
\end{note}
\begin{example}
Solve $\sqrt[3]{x^2+12x+19}=x+1$.

\end{example}
