\subsection{Radical Equations with No Solution} 
An equation involving a function might have no solution due to the limited \makeblank[1.5in] of the function.
\begin{itemize}
\item If $n$ is \makeblank[1in], the range of $\sqrt[n]{x}$ is $\R$
\item If $n$ is \makeblank[1in], the range of $\sqrt[n]{x}$ is $\Set{y|\makeblank[1.5in]}$
\end{itemize}
So the equation $\sqrt[n]{X}=c$ has no solution if $n$ is \makeblank[1in] and $c$ is \makeblank[1.5in].
\begin{example}
Solve each radical equation, if it has solutions.
\begin{lpicvsplit}{2}{3}
\foreach \x/\eqn in {0/{\sqrt[3]{x}=5},1/{\sqrt[4]{x}=2}}
	\regtext{\x*\value{fcols}+\x+0.5*\value{fcols}}{-1}{$\eqn$};
\end{lpicvsplit}
\begin{lpicvsplit}{2}{3}
\foreach \x/\eqn in {0/{\sqrt[5]{x}=-3},1/{\sqrt{x}=-7}}
	\regtext{\x*\value{fcols}+\x+0.5*\value{fcols}}{-1}{$\eqn$};
\end{lpicvsplit}
\end{example}
\noi As usual, we need to check this \emph{after} isolating the radical.
\begin{example}
Solve each radical equation, if it has a solution.
\begin{lpicvsplit}{2}{9}
\foreach \x/\eqn in {0/{-2\sqrt{3x-1}+5=-3},1/{3\sqrt{4x+1}+7=4}} {
	\regtext{\x*\value{fcols}+\x+0.5*\value{fcols}}{-1}{$\eqn$};
	\regtextleft{\x*\value{fcols}+\x}{-8}{Check:};
}
\end{lpicvsplit}
\end{example}
