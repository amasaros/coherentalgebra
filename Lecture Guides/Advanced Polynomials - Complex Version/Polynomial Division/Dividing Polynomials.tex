\subsection{Dividing Polynomials}
Polynomials are like numbers, with the terms taking the place of digits. So if we want to divide
\[
\left(4x^4 - 4x^3 - x^2 + 13x - 12\right)\div \left(2x^2 + x - 3\right)
\]
we follow the same steps as for long division.
