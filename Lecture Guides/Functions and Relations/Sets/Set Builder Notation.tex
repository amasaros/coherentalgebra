\subsection{Set Builder Notation} 
We have an alternate way of describing the elements of a set.
\begin{definition}[Set Builder Notation]
\term{Set Builder Notation} is a method of describing a set in terms of a generic element, followed by a description of rules the elements must follow. Set Builder Notation looks like this:
\[
\Set{\mbox{generic element}\mid\mbox{rules}}
\]
\end{definition}

\item How else could we describe $A$?\makeblanknew
\end{itemize}
\end{Example}
