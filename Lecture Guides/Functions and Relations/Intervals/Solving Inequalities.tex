\subsection{Solving Inequalities} 
The properties of inequality give us basic moves to solve.
\begin{itemize}
\item We can \makeblank[1in] the same number to each side of an inequality.
\item We can \makeblank[1.5in] by the same number on each side of an equation.
\end{itemize}
In inequalities, order matters. We must reverse the direction when we:
\begin{itemize}
\item multiply or divide by a \makeblank[1.5in] number.
\item switch the \makeblank[1.5in] of the inequality.
\end{itemize}
