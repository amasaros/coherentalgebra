\subsection{Basic Absolute Value Equations} 
Here is the graph of $y=\Abs{x}$.
\begin{center}
\begin{tikzpicture}
\setgraph{4}{4}{1}{4}
\setspace{1}
\GraphLarge[1]
\draw[graph] (-4,4) -- (0,0) -- (4,4);
\end{tikzpicture}
\end{center}
This function fails the \makeblank[3in], so it is not\linebreak\makeblank. In general, each $y$ comes from \makeblanksmall values of $x$.
\begin{Example}Solve each equation.
\begin{itemize}
\item $\Abs{x}=4$\hfill$x=\makeblanksmall$ or $x=\makeblanksmall$
\item $\Abs{x}=3$\hfill$x=\makeblanksmall$ or $x=\makeblanksmall$
\item $\Abs{x}=0$\hfill$x=\makeblanksmall$
\item $\Abs{x}=-4$\hfill\makeblank
\end{itemize}
\end{Example}
\begin{Note}The presence of a function that is not \makeblank[1in] means that in general we will have \makeblanksmall solutions. The presence of the function whose range is not \makeblank means that we will sometimes have \makeblanksmall solutions.\end{Note}
