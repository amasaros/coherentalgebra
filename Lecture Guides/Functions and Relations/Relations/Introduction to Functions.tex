\subsection{Introduction to Functions} 
We are especially interested in a specific type of relation.
\begin{definition}[Function]
We say a relation $R$ is a \term{function} when each element $x$ of the domain of $R$ appears in only one pair $(x,y)\in R$.
\end{definition}
We can describe this as ``each element of the domain only appears once'' or ``each $x$ is related to at most one $y$.''
\begin{Example}
For each relation below, determine whether the relation is a function. If it is not a function, list two ordered pairs that show that it is not a function.
\begin{itemize}
\item $R=\Set{(-9,2),(1,-4),(2,-9),(5,0)}$\makeblanknew
\item $S=\Set{(-2,-7),(-2,7),(0,-1),(4,6)}$\makeblanknew
\item $T=\Set{(-8,-7),(-3,-2),(-3,5),(5,-2)}$\makeblanknew
\item $Q=\Set{(-8,-9),(-3,0),(6,9),(8,9)}$\makeblanknew
\end{itemize}
\end{Example}
