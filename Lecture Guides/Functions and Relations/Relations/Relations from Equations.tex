\subsection{Relations from Equations} 
We will be most interested in relations expressed using algebraic equations. We can express such relations in set-builder notation.
\begin{example}
Let $R=\Set{(x,y)\mid x^2+y^2=25}$. For each statement below, determine whether the statement is true or false.
\begin{lpicvsplit}{3}{4}
\foreach \x/\q in {0.5*\value{fcols}/-5\rel{R}0,1.5*\value{fcols}+1/-3\rel{R}4,2.5*\value{fcols}+2/5\rel{R}5} {
	\regtext{\x}{-1}{$\q$};
}
%\foreach \x in {0.5*\value{cols},\value{cols},1.5*\value{cols}} \draw (\x,0) -- (\x,-5);
\end{lpicvsplit}
\end{example}
\begin{example}
Let $S=\Set{(x,y)\mid 3x+2y=12}$. Fill in the blanks below to make the statements true.
\begin{lpicvsplit}{3}{6}
\foreach \x/\q in {0.5*\value{fcols}/2\rel{S}\makespace[\widthof{2}],
									1.5*\value{fcols}+1/\makespace[\widthof{0}]\rel{S}0,
									2.5*\value{fcols}+2/0\rel{S}\makespace[\widthof{0}]}
	\regtext{\x}{-1}{$\q$};
\end{lpicvsplit}
\end{example}
