\subsection{One to One Relations} 
\begin{definition}[One-to-One]
We say a relation $R$ is \term{one-to-one} when each element $y$ of the range of $R$ appears in only one pair $(x,y)\in R$.
\end{definition}
We can describe this as ``each element of the range only appears once'' or ``each $y$ is related to at most one $x$.''
\begin{Note}``One-to-one'' is an adjective. So we can talk about a one-to-one function, or about a one-to-one relation, but we can't talk about ``a one-to-one.''\end{Note}
\begin{Example}
For each relation below, determine whether the relation is one-to-one. If it is not one-to-one, list two ordered pairs that show that it is not.
\begin{itemize}
\item $R=\Set{(-9,2),(1,-4),(2,-9),(5,0)}$\makeblanknew
\item $S=\Set{(-2,-7),(-2,7),(0,-1),(4,6)}$\makeblanknew
\item $T=\Set{(-8,-7),(-3,-2),(-3,5),(5,-2)}$\makeblanknew
\item $Q=\Set{(-8,-9),(-3,0),(6,9),(8,9)}$\makeblanknew
\end{itemize}
\end{Example}
