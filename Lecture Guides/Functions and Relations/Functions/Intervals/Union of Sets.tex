\subsection{Union and Intersection of Sets} 
We can combine sets in several ways.
\begin{definition}[Union]
The \term{union} of two sets is the set which contains all elements in each set. That is, for sets $A$ and $B$:
\[
A\cup B=\{x|x\in A\mbox{ or }x\in B\}
\]
The symbol $\cup$ represents the union. $A\cup B$ is read ``$A$ union $B$.''
\end{definition}
\begin{note}
In mathematics, when we say ``or,'' we mean an \emph{inclusive} or; that is, one or the other or both.
\end{note}
\begin{example}
Let
\[
A=\{2,4,6,8,10,12\}
\qquad
B=\{3,6,9,12\}
\]
Then $A\cup B=\makeblank[3in]$
\end{example}
\begin{definition}[Intersection]
The \term{intersection} of two sets is the set which contains elements that are in \emph{both} sets. That is, for sets $A$ and $B$:
\[
A\cap B=\{x|x\in A\mbox{ and }x\in B\}
\]
The symbol $\cap$ represents the intesection. $A\cap B$ is read ``$A$ intersect $B$.''
\end{definition}
\begin{example}
Let
\[
A=\{2,4,6,8,10,12\}
\qquad
B=\{3,6,9,12\}
\]
Then $A\cap B=\makeblank[1.5in]$
\end{example}
It may be that we take the intersection of two sets, and there is nothing there.
\begin{definition}[Empty Set]
An \emph{empty set}, denoted $\emptyset$ or $\{\}$, is a set which contains no elements.
\end{definition}
\begin{example}
Let
\[
A=\{1,2,4,8\}
\qquad
B=\{10,20,30\}
\]
Then $A\cap B$ contains \makeblank, so $A\cap B=\makeblanksmall$.
\end{example}
