\subsection{Solution Sets} 
\begin{definition}[Solution Set]
The \term{solution set} of an equation or an inequality is the set which contains all and only the solutions of the equaiton or inequality.
\end{definition}
For equations, the solution set is often very boring.
\begin{Example}
Write the solution set of $2x-3=7$, and graph this solution set on a number line.
\begin{cpic}{}{6}
\regtextleft{1-\value{cols}}{-4}{Solution set:};
\regtextleft{1-\value{cols}}{-5}{Graph:};
\draw[axis] (-5,-5) -- (5,-5);
\foreach \x in {-4,...,4} \draw[tick] (\x,-5+0.2) -- (\x,-5-0.2);
\end{cpic}
\end{Example}
For inequalities, this is more interesting.
\begin{Example}
Write the solution set of $5-2x>7$, and graph this solution set on a number line.
\begin{cpic}{}{6}
\regtextleft{1-\value{cols}}{-4}{Solution set:};
\regtextleft{1-\value{cols}}{-5}{Graph:};
\draw[axis] (-5,-5) -- (5,-5);
\foreach \x in {-4,...,4} \draw[tick] (\x,-5+0.2) -- (\x,-5-0.2);
\end{cpic}
\end{Example}
