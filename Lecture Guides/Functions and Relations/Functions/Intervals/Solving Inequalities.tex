\subsection{Solving Inequalities} 
\begin{displaybox}[Solve Any Inequality]
When we solve an inequality, we use the same steps as we would to solve the corresponding \makeblank[1.5in], except:
\begin{itemize}
\item If we \makeblank[1in] the sides, we must reverse the inequality.
\item If we \makeblank[1.5in] both sides by a \makeblank[1.5in] number, we must reverse the inequality.
\item Be aware of the structures behind more advanced moves.
\end{itemize}
\end{displaybox}
