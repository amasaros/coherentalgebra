\subsection{Solving Compound Inequalities} 
To solve a compound inequality:
\begin{enumerate}
\item Solve each inequality separately.
\item Sketch the solution set.
\item Write the solution.
\end{enumerate}
\begin{example}
Solve $3x-5<18$ and $2x+6>1$
