\subsection{Adding and Subtracting Rational Expressions} 
To add or subtract rational expressions with different denominators, we follow the same steps as we do to add or subtract fractions of numbers:
\begin{procedure}{Adding or Subtracting Fractions}
\begin{enumerate*}
\item Find the least common multiple of the denominators (LCD).
\item Write equivalent fractions with that denominator.
\item Add the new fractions (that have the same denominator).
\item Simplify, if necessary.
\end{enumerate*}
\end{procedure}
\begin{Example}
Add $\dfrac{x+4}{2x^2+x-6}+\dfrac{3x+1}{2x^2-13x+15}$. Simplify your answer as much as possible.

\end{Example}
