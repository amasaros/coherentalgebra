\subsection{Adding and Subtracting with Like Denominators} 
As with fractions, we can add and subtract rational expressions with the same \makeblank[1in] by adding or subtracting the \makeblank[1in].
\begin{example}
Add or subtract. Simplify your answer as much as possible.
\begin{lpic}{}{19}
\foreach \y/\l/\na/\nb/\d/\s in {1/a/3x/4x/5y/+,
																3/b/11x^2/7/4x+1/-,
																5/c/x^2-7x+2/x^2+2x-14/6x^2-x-15/+,
																14/d/x^2+6x+9/12x/x^4-81/-} {
	\fractextright{1}{-\y}{\l. };
	\fractextleft{1}{-\y}{$\dfrac{\na}{\d}\s\dfrac{\nb}{\d}={}$};
}
\end{lpic}
\end{example}
