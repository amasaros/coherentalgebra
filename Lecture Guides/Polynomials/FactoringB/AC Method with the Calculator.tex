\subsection{AC Method with the Calculator} 
When using the AC Method, we often need to find all factors of a large number with many factors.
\begin{Example}
Use the AC Method to factor $6x^2+23xy+20y^2$ using the AC Method.\lbr
We must find two numbers whose product is \makeblank[0.75in] and whose sum is \makeblanksmall. This requires we find all factors of \makeblanksmall!
\begin{lpic}{}{8}
\foreach \y/\p/\q/\s in {1/1/120/121,
												2/2/60/62,
												3/3/40/43,
												4/4/30/34,
												5/5/24/29,
												6/6/20/26,
												7/8/15/23,
												8/10/12/22} {
		\regtextright{1}{-\y}{$\p$};
		\regtext{1.5}{-\y}{$\times$};
		\regtextright{3}{-\y}{$\q$};
		\regtext{3.5}{-\y}{$=$};
		\regtextright{5}{-\y}{$120$};
		\regtextright{7}{-\y}{$\p$};
		\regtext{7.5}{-\y}{$+$};
		\regtextright{9}{-\y}{$\q$};
		\regtext{9.5}{-\y}{$=$};
		\regtextright{11}{-\y}{$\s$};
	}
\end{lpic}
\end{Example}
\noindent This is a lot of factors! Whenever we want to do highly repetative arithmetic, we want a \makeblank[1in] to do it for us. The procedure here uses the TI-84 calculator, but it can be done with many other calculators or with a computer.
\begin{procedure}{Factors for the AC Method on the TI-84}
\begin{enumerate*}
\item Press \TIbut{y=} to access the function entry screen.
\item Enter \TI{\#\#/X} as \TI{Y\TIsub{1}} and \TI{\#\#/X+X} as \TI{Y\TIsub{2}}, where \TI{\#\#} is the desired product.
\item Press \TIbut{2nd}\ \TIbut{graph} to access the ``Table'' feature.
\item The \TI{X} and \TI{Y\TIsub{1}} columns show the two factors, and the \TI{Y\TIsub{2}} column shows their sum. Use the up and down arrow keys to scroll through the table until the desired sum is found.
\end{enumerate*}
\end{procedure}
