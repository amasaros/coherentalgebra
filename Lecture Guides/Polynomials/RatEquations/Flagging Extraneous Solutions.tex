\subsection{Flagging Extraneous Solutions} 
There are two ways we can control extraneous solutions.
\begin{enumerate*}
\item When we know one of our steps might have caused an extraneous solution, we can check our solutions in the original equation.
\item When we perform a step that might cause an extraneous solution, we can ``flag'' the forbidden values.
\end{enumerate*}
Let's see an example with the second method.
\begin{Example}
Solve $\dfrac{8x-10}{x^2-3x+2}-\dfrac{x+29}{x^2+3x-4}=\dfrac{8x+2}{x^2+2x-8}$ by clearing denominators.
\begin{lpic}{}{23}
\foreach \y/\l/\inst in {1/a/Find the LCD,4/b/Multiply both sides by,10/c/Solve the new equation,19/d/Check}
{
	\regtextright{1}{-\y}{\l. };
	\regtextleft{1}{-\y}{\inst};
}
\regtextleft{1}{-5}{$x\neq{}$};
\foreach \x/\y/\exp in {1/-2/x^2-3x+2,\value{cols}/-2/x^2+3x-4,1/-3/x^2+2x-8,\value{cols}/-3/\mathrm{LCD}}
	\regtextleft{\x}{\y}{$\exp={}$};
\end{lpic}
\end{Example}
