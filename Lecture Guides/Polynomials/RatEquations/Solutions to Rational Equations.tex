\subsection{Solutions to Rational Equations} 
\begin{definition}[Rational Equation]
A \term{rational equation} is an equation in which both sides are rational expressions, at least one of which has a variable in the denominator.
\end{definition}
\begin{Example}
$\dfrac{3x-1}{x-2}=4$ is a rational equation. Determine whether each value below is a solution to the equation:
\begin{enumerate}\setabc
\item\makeblankfill{$x=0$ }
\item\makeblankfill{$x=3$ }
\item\makeblankfill{$x=7$ }
\end{enumerate}
\end{Example}
\begin{Example}
$\dfrac{2x-1}{x-3}=\dfrac{3x+5}{x+1}$ is a polynomial equation. Determine whether each value below is a solution to the equation:
\begin{enumerate}\setabc
\item\makeblankfill{$x=-2$ }
\item\makeblankfill{$x=0$ }
\item\makeblankfill{$x=2$ }
\item\makeblankfill{$x=7$ }
\end{enumerate}
\end{Example}
\noi Note that rational equations typically have \makeblank solution.
