\subsection{Clearing Denominators} 
We can \makeblank[1.5in] both sides of an equation by the LCD to ``clear denominators.''
\begin{example}
Solve $\frac{2}{3}x-\frac{4}{5}=\frac{3}{5}x+\frac{8}{15}$ by first clearing denominators.
\begin{lpic}{}{9}
\foreach \y/\l/\inst in {1/a/Find the LCD,3/b/Multiply both sides by,6/c/Solve the new equation,9/d/Check}
{
	\regtextright{1}{-\y}{\l. };
	\regtextleft{1}{-\y}{\inst};
}
\end{lpic}
\end{example}
We can do the same thing when the denominators contain variables.
\begin{example}
Solve $\dfrac{x^2-10}{x^2}=\dfrac{3}{x}$.
\begin{lpic}{}{16}
\foreach \y/\l/\inst in {1/a/Find the LCD,3/b/Multiply both sides by,8/c/Solve the new equation,13/d/Check}
{
	\regtextright{1}{-\y}{\l. };
	\regtextleft{1}{-\y}{\inst};
}
\end{lpic}
\end{example}
