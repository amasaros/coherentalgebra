\subsection{Exponent Rules (Negative Exponents and Examples)}
We define the remaining integer exponents to be consistent with the product and quotient rules.
\begin{definition}[Exponent Zero]
For any real number $x\neq 0$, we define

\nic$x^0=1$
\end{definition}
\begin{itemize}
\item This is consistent with the product rule:\lbr
$x^0x^n=x^{0+n}=\makeblanksmall=\makeblanksmall\cdot x^n$
\item This is consistent with the quotient rule (if we allow $m=n$):\lbr
$\dfrac{x^n}{x^n}=x^{n-n}=\makeblanksmall$, but $\dfrac{x^n}{x^n}=\makeblanksmall$
\end{itemize}
\begin{definition}[Negative Integer Exponents]
For any real number $x\neq 0$ and positive integer $n$,$$x^{-n}=\dfrac{1}{x^n}$$
\end{definition}
This lets us simplify the statement of the quotient rule:
\begin{theorem}[Quotient of Powers Rule]
For any real number $x$ and integers $m$ and $n$,
$$
\dfrac{x^m}{x^n}=x^{m-n}=\dfrac{1}{x^{n-m}}
$$
\end{theorem}
