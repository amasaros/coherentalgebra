\subsection{Review: Exponents} 
\begin{definition}
We represent repeated multiplication using \term{positive integer exponents}. The expression $b^p$ is defined as
\[
b^p=\underbrace{b\cdot b \cdots b}_{\mbox{$p$ copies}}
\]
The number $b$ in this expression is called the \term{base}, and the number $p$ is called the \term{exponent}.
\end{definition}
