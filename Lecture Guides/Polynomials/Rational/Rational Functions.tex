\subsection{Rational Functions} 
\begin{definition}[Rational Function]
We can define a rational function in two ways.
\begin{itemize}
\item A rational function is one whose expression is a rational expression.
\item A rational function is one that can be found by dividing two polynomial functions, that is, if $P$ and $Q$ are polynomial functions, $R=\dfrac{P}{Q}$ is a rational function.
\end{itemize}
\end{definition}
\begin{Example}
Let $R(x)=\dfrac{2x^2-7x+5}{3x^2-7x+10}$. Evaluate each of the following, if defined.

As we study the arithmetic of rational expressions, we will discover the following fact.
\begin{theorem}[Closure of Rational Functions]
The set of rational functions is \term{closed} under addition, subtraction, multiplication, and division. That is, if $R$ and $S$ are rational functions, $R+S$, $R-S$, $R\cdot S$, and $\dfrac{R}{S}$ are also rational functions.
\end{theorem}
