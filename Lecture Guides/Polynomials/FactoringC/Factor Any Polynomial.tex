\subsection{Factor Any Polynomial} 
Follow these steps to factor any polynomial.
\begin{enumerate}
\item Factor out the GCF.
\item The next step depends on the number of terms:
\begin{description}
\item [Two Terms] Can only factor if it's a special form.
\begin{enumerate}
\item For degree 2 or 3, anything other than $A^2-B^2$, $A^3-B^3$, or $A^3+B^3$ is prime.
\item For degree 4+, may need extra special forms; if this fails, turn to advanced techniques.
\end{enumerate}
\item [Three Terms] Try these things.
\begin{enumerate}
\item If it's quadratic, use the AC Method. If this fails, it's prime.
\item If it's higher degree, try to pretend it's quadratic and use the AC Method. If this fails, turn to advanced techniques.
\end{enumerate}
\item [Four Terms] Try these things.
\begin{enumerate}
\item Factor by grouping. If this fails and it's quadratic, can't factor as product of binomials.
\item If grouping fails, try the difference of squares trick.
\item If both fail, turn to advanced techniques.
\end{enumerate}
\item [Five+ Terms] Turn to advanced techniques.
\end{description}
\item After factoring, return to step 2 with each factor.
\end{enumerate}
