\subsection{Adding Polynomials} 
Adding polynomials is an application of the associative and commutative property.
\begin{example}Add $(2x^2-5x+6)+(3x^2-11x-5)$, justifying each step.
\why{8}
\end{example}
\noindent We will typically not write all of these steps. If you remember that \makeblank[1in] signs go with their term, you may be able to directly write down the sum.
