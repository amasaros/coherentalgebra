\subsection{Factoring by Grouping Rectangle Method} 
We can also perform factoring by grouping by running the rectangle method of multiplication backward. This can make it easier to keep track of signs.
\begin{example}
Factor $p^2+3ps-2pq-6qs$ by grouping, using the rectangle method.
\begin{lpic}{}{5}
\draw (2,-1) rectangle (6,-3);
\draw (2,-2) -- (6,-2);
\draw (4,-1) -- (4,-3);
\end{lpic}
\end{example}
